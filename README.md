## Component List

* 1x ATMEGA328 Microcontroller
* 2x 74HC595 Shift Register
* 2x Common Cathode 7 Segment Display
* 1x LM7805 5v Linear Regulator
* 1x LD1117V33 3.3v Linear Regulator
* 1x DC Barrel Jack
* 1x 5mm LED
* 1x 2 Pin Push Button
* 2x 10uF Capacitor
* 1x 1uF Capacitor
* 1x 0.1uF Capacitor
* 2x 22pF Capacitor
* 1x 16MHz Crystal
* 3x 10k Ohm Resistor
* 4x 470 Ohm Resistor
* 1x 1k Ohm Resistor
